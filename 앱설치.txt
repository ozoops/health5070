새로운 팀원이 이 프로젝트를 자신의 컴퓨터에서 실행하기 위한 전체 과정은 다음과 같습니다.

   1. 프로젝트 내려받기 (Git Clone)
      먼저 팀원이 자신의 컴퓨터에 Git으로 프로젝트를 복제해야 합니다. (GitHub에서 URL을 복사해서 사용합니다.)
   1     git clone [GitHub 저장소 URL]

   2. 가상 환경 생성 및 활성화
      프로젝트 폴더로 들어간 후, 파이썬 가상 환경을 만들어 프로젝트의 라이브러리들이 다른 프로젝트와 섞이지 않도록
  격리합니다.

   1     # 가상 환경 생성 (최초 1회)
   2     python -m venv venv
   3
   4     # 가상 환경 활성화 (터미널을 열 때마다 실행)
   5     .\venv\Scripts\activate

   3. 필요한 라이브러리 설치
      requirements.txt 파일에 기록된 모든 라이브러리를 설치합니다.

   1     pip install -r requirements.txt

   4. 환경 변수 설정 (.env 파일 생성)
      이 부분이 중요합니다. API 키 같은 민감한 정보는 Git에 올라가지 않습니다. 프로젝트 최상위 폴더에 .env 라는
  파일을 직접 만들고, 그 안에 다음과 같이 필요한 키를 적어주어야 합니다.
   1     OPENAI_API_KEY="sk-..."

   5. 앱 실행
      모든 설정이 끝나면, 다음 명령어로 Streamlit 앱을 실행할 수 있습니다.

   1     streamlit run app.py

  이 과정을 거치면 모든 팀원이 각자 자신의 컴퓨터에서 동일한 개발 환경을 구축하고 작업을 시작할 수 있습니다.
